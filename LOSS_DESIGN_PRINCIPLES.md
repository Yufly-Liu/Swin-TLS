# 损失函数设计原理详解

## 🎯 问题本质

### 场景分析
```
真值图像：
- 黑色背景：90% 像素，值 ≈ 0.0
- 白色线条：10% 像素，值 ≈ 1.0
```

### 标准 MSE 的问题

```python
Loss = mean((pred - target)²)
```

**数值示例**：
```
假设 512×512 图像 = 262,144 像素
- 背景像素：236,000 个（90%）
- 线条像素：26,144 个（10%）

预测1（全黑，丢失线条）：
  背景误差 = (0 - 0)² = 0
  线条误差 = (0 - 1)² = 1
  Loss = (0×236000 + 1×26144) / 262144 = 0.0997

预测2（保留线条）：
  背景误差 = (0 - 0)² = 0
  线条误差 = (0.9 - 1)² = 0.01
  Loss = (0×236000 + 0.01×26144) / 262144 = 0.00099

差异 = 0.0997 - 0.00099 = 0.0987
```

看起来有差异，但在实际训练中，模型会发现：
- 把白斑变黑（背景）：loss 下降很快
- 恢复线条：loss 下降很慢

结果：模型优先学习去除白斑，忽略线条恢复。

## 1️⃣ WeightedMSELoss（加权 MSE）

### 核心思想
**给重要的像素（线条）更高的权重**

### 数学公式
```python
# 步骤1：创建权重图
target_gray = mean(target, dim=channel)  # 转灰度
weight_map = where(
    target_gray > threshold,
    foreground_weight,  # 线条区域：15.0
    1.0                 # 背景区域：1.0
)

# 步骤2：加权 MSE
Loss = mean(weight_map × (pred - target)²)
```

### 数值示例（weight=15）
```
预测1（全黑，丢失线条）：
  背景误差 = 0² × 1.0 = 0
  线条误差 = 1² × 15.0 = 15.0
  Loss = (0×236000 + 15.0×26144) / 262144 = 1.496

预测2（保留线条）：
  背景误差 = 0² × 1.0 = 0
  线条误差 = 0.01² × 15.0 = 0.0015
  Loss = (0×236000 + 0.0015×26144) / 262144 = 0.00015

差异 = 1.496 - 0.00015 = 1.496（提升 15 倍！）
```

### 为什么有效？
线条区域的误差被放大 15 倍，模型的梯度主要来自线条区域，被迫学习线条恢复。

## 2️⃣ EdgePreservingLoss（边缘保持）

### 核心思想
**明确优化边缘清晰度**

### 数学公式
```python
# 步骤1：使用 Sobel 算子检测边缘
Sobel_x = [[-1, 0, 1],
           [-2, 0, 2],
           [-1, 0, 1]]

Sobel_y = [[-1, -2, -1],
           [ 0,  0,  0],
           [ 1,  2,  1]]

edge_x = conv2d(image, Sobel_x)
edge_y = conv2d(image, Sobel_y)
edge_strength = sqrt(edge_x² + edge_y²)

# 步骤2：组合损失
Loss = MSE(pred, target) + edge_weight × MSE(edge(pred), edge(target))
```

### 为什么有效？
- MSE 只关注像素值，不关注结构
- 边缘损失明确优化边缘强度和位置
- 线条的边缘被明确保护

## 3️⃣ FocalMSELoss（Focal 损失）

### 核心思想
**对误差大的像素（难样本）赋予更高权重**

### 灵感来源
借鉴目标检测中的 Focal Loss（Lin et al., 2017）：
- 简单样本（背景）：权重低
- 困难样本（细节）：权重高

### 数学公式
```python
# 步骤1：计算 MSE
mse = (pred - target)²

# 步骤2：归一化到 [0, 1]
mse_normalized = mse / (max(mse) + ε)

# 步骤3：计算 Focal 权重
focal_weight = (1 - exp(-mse_normalized))^gamma

# 步骤4：加权
Loss = mean(mse × (1 + focal_weight))
```

### 为什么有效？
```
误差小的像素（背景，已经学好）：
  mse = 0.001
  focal_weight ≈ 0
  实际权重 = 1 + 0 = 1

误差大的像素（线条，还没学好）：
  mse = 0.5
  focal_weight ≈ 2
  实际权重 = 1 + 2 = 3
```

模型自动关注还没学好的区域（线条）。

## 4️⃣ CombinedLoss（组合损失）⭐ 最强

### 核心思想
**三管齐下，全方位优化**

### 数学公式
```python
Loss = WeightedMSE(pred, target) 
     + edge_weight × EdgeLoss(pred, target)
     + 0.5 × FocalLoss(pred, target)
```

### 为什么最强？
1. **WeightedMSE**：强制关注线条区域（空间权重）
2. **EdgeLoss**：明确优化边缘清晰度（结构约束）
3. **FocalLoss**：自动识别难样本（自适应权重）

三种策略互补，覆盖所有问题。

### 测试结果
```
标准 MSE：    差异 0.116（基线）
加权 MSE：    差异 1.200（10倍提升）
组合损失：    差异 2.920（25倍提升）⭐
```

## 5️⃣ PerceptualWeightedLoss（感知加权）

### 核心思想
**自适应识别重要区域**

### 数学公式
```python
# 步骤1：计算每张图的亮度阈值（前 10% 最亮）
threshold = quantile(target, 90%)

# 步骤2：线性映射权重
weight_map = where(
    target > threshold,
    min_weight + (max_weight - min_weight) × (target - threshold) / (1 - threshold),
    min_weight
)

# 步骤3：加权 MSE
Loss = mean(weight_map × (pred - target)²)
```

### 为什么有效？
- 不需要手动设置阈值
- 根据每张图的亮度分布自动调整
- 适合亮度变化大的数据集

## 🔧 实现细节

### 数值稳定性

所有损失函数都包含数值稳定性保护：

```python
# 1. 避免除零
denominator = value + 1e-8

# 2. 避免 sqrt(负数)
value = max(0, value)
result = sqrt(value)

# 3. 避免 log(0)
result = log(value + 1e-8)

# 4. 梯度裁剪
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
```

### 计算效率

- WeightedMSE：+5% 计算时间
- EdgePreserving：+15% 计算时间（Sobel 卷积）
- FocalMSE：+10% 计算时间
- Combined：+20% 计算时间

权衡：效果提升 >> 计算开销

## 📊 对比总结

| 损失函数 | 原理 | 优势 | 劣势 | 适用场景 |
|---------|------|------|------|---------|
| 标准 MSE | 平等对待所有像素 | 简单快速 | 忽略少数类 | 平衡数据 |
| 加权 MSE | 空间权重 | 直接有效 | 需要设置阈值 | 类别不平衡 |
| 边缘保持 | 结构约束 | 边缘清晰 | 计算稍慢 | 细节模糊 |
| Focal | 自适应权重 | 自动识别难样本 | 需要调参 | 难易样本混合 |
| 组合 | 三管齐下 | 最强效果 | 计算最慢 | 严重不平衡 |
| 感知加权 | 自适应阈值 | 无需手动调参 | 可能不够激进 | 亮度不一致 |

## 🎓 理论基础

### 1. 类别不平衡问题
- 来源：语义分割、目标检测
- 解决：重采样、重加权、Focal Loss

### 2. 感知损失
- 来源：风格迁移、超分辨率
- 解决：使用预训练网络提取特征

### 3. 边缘保持
- 来源：图像处理、去噪
- 解决：Total Variation、Sobel 算子

### 4. 难样本挖掘
- 来源：目标检测（Focal Loss）
- 解决：动态调整样本权重
